## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Build and run the production server:

```bash
npm run build
npm run start
```

## Customize Content

Edit the data file and images:

- Data: `src/data/data.tsx`
- Images: `src/images/`
- Resume PDF: `public/Alireza-Rahimpanah-main.pdf`

## Deployment

Deploy with Vercel or any Next.js-compatible host. For Vercel, import the repo and follow the default Next.js settings.
